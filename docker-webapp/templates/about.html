{% block title %} About Unbrickable {% endblock %}

Unbrickable kalkuliert potenzielle Gewinne, die durch den Kauf von Einzelteilen einer Minifigur und den Verkauf der Figur entstehen können. Die Daten werden mittles API-Abfrage in eine SQL-Datenbank geschrieben, verarbeitet und die Ergebnisse in einer Flask-Webapp zur Verfügung gestellt.
